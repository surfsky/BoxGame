# 推箱子游戏

基于Trae、phaserjs、nodejs、typescript、vite、豆包等技术创建
在AI的帮助下，核心代码花了半天时间完成。

Author：github.com/surfsky
Last Update：2025-05




## 功能

- 经典推箱子游戏改版；
- 增加冰块地面，物体和角色会滑动，增加游戏难度和趣味性；
- 支持虚拟按键、键盘操作；
- 支持摄像头体感操作：
    右：右手平举
    左：左手平举
    上：双手上举
    下：下蹲
    ok: 敬礼或者奥特曼
- 支持AI自动寻路；
